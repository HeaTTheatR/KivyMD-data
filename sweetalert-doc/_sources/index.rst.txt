Welcome to Sweet Alert's documentation!
=======================================

.. autoapimodule:: sweetalert

Beautiful, responsive, customizable and accessible popup replacement
for `KivyMD <https://github.com/kivymd/KivyMD>`_ library.

.. image:: https://github.com/HeaTTheatR/KivyMD-data/raw/master/gallery/preview-sweet-alert.gif
    :align: center

Installation
------------

.. code-block:: bash

    pip install kivymd-components

and

.. code-block:: bash

    componemts install sweetalert
